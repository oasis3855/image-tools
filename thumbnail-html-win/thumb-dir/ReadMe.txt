１．動作環境
     Windows NT, 2000 （日本語版)
     Windows 95, 98, 98SE, Me (日本語版)
     MFC42.DLLおよびMSVCRT.DLLがシステムフォルダに存在すること


     このプログラムに、縮小画像を作る機能はありません。別途縮小画像を作成で
     きるプログラムを用意してください。

２．インストール
     配布されたプログラムを、適当なフォルダにコピーする。
     ★このプログラムは、レジストリや設定ファイル類を一切利用しません。

３．ThumbHtmlImg の利用法
     このプログラムは、ある一つのフォルダにある画像ファイルの一覧表示をする
     ためのHTMLファイルを作成します。
     ThumbHtmlImgを実行する前に、画像ファイルと、その縮小イメージの入ったフ
     ォルダを用意してください。双方のフォルダは「基準フォルダ｣の下の「子フォ
     ルダ｣としてください。
     画像フォルダの対象拡張子は、複数入力できます。セミコロン「；｣で区切って
     入力してください。
     サムネイル画像の対象拡張子は、１種類しか指定できません。

     ダイアログ上で必要な設定を行ったら、「処理実行｣ボタンを押してください。

４．ThumbHtmlDir の利用法
     このプログラムは、複数のフォルダに分かれて格納されている画像ファイルの
     一覧表示をするためのHTMLファイルを作成します。
     ダイアログ上で必要な設定を行ったら、「処理実行｣ボタンを押してください。
     「子フォルダ｣の下の画像ファイルの最初の一つ（名前順の並べ替えで、若い物）
     をサムネイル用として「サムネイルフォルダ｣に名前を変えてコピーします。
     ●重要● サムネイルフォルダにコピーされた画像は、別途、縮小処理を行って
     ください。

５．ThumbHtmlImgのダイアログの説明
     「基準フォルダ｣はHTMLファイルを作成するフォルダです。また、画像フォルダ
     とサムネイル画像フォルダはこの「基準フォルダ｣からの相対指定のフォルダと
     なります。

     「画像フォルダ｣は、画像の入ったフォルダです。

     「サムネイル画像フォルダ｣は、上記の画像フォルダの画像を縮小したファイル
     が入ったフォルダです。「画像フォルダ｣と「サムネイル画像フォルダ｣のそれ
     ぞれのファイル名は同一名である必要があります。（拡張子を除く） 名前が同
     じで、拡張子だけが違うファイルが存在する場合の処理は、不定となります。

     「HTMLファイル名｣は出力するHTMLファイル名を指定します。

     「HTML表の列数｣は、横に何枚の画像を並べて表示するかを設定します。

     「IMGタグのオプション｣と「A LINKタグのオプション｣は、タグ内に設定するオ
     プションを指定します。

６．ThumbHtmlDirのダイアログの説明
     「親フォルダ｣はHTMLファイルを作成するフォルダであり、画像が入っている子
     フォルダの親フォルダです。

     「サムネイル画像フォルダ｣は縮小画像ファイルを格納するフォルダを指定しま
     す。

     「HTML表の列数｣は、横に何枚の画像（フォルダ表示）を並べるかを設定します。

     「HTMLファイル名｣は出力するHTMLファイル名を指定します。

７．アンインストール
     インストール時にコピーしたファイルを消去するだけです。
