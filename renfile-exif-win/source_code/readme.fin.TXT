【はじめに】
このプログラムは、EXIFデータに基づいてファイル名とタイムスタンプを変換す
るツールです


【動作環境】
Windows 95/98/Me/NT/2000/XP/2003/Vista (日本語版および各国語版)
MFC71.DLL および MSVCRT71.DLL がシステムフォルダに存在すること（Visual C
++ 7.1 対応版） 古いバージョンのDLLの場合、起動できません。


このプログラムには、日本語と英語の２種類の設定が含まれています。日本語以
外のWindowsで起動した場合、自動的に英語モードで動作します。


【インストール】
プログラムを任意のフォルダにコピーして、実行するだけです。


【アンインストール】
このプログラムは、レジストリおよび設定ファイルなどを利用しません。
プログラム本体を消去すれば、アンインストール完了です。


【利用法】
ファイル名・タイムスタンプをを変換したいファイルを「ファイル参照」ボタン
で1つ指定します。または「フォルダ参照」ボタンでフォルダごと指定することが
できます。

ファイル指定の後、実行する変換モード「ファイル名の変換」または「タイムス
タンプの変更」を選択して、実行ボタンを押します。
なお、変換モードを「EXIFデータの表示」とすると、EXIFデータを読み出して表
示するだけという動作を行い、ファイル名やタイムスタンプの変更を行いません。
